# Handoff Screenshot Requirements

## Required 6 screenshots
1. Desktop Idle (1440)
2. Desktop Loading (1440)
3. Desktop InFocus (1440)
4. Desktop Empty (1440)
5. Mobile Idle (390)
6. Mobile InFocus (390)

## Naming convention
- focushub_<state>_<breakpoint>_<date>.png
- Example: focushub_idle_1440_2026-04-20.png

## Required notes for each screenshot
- Primary visual focus
- Available interactions in this state
- Disabled/locked controls in this state

## Automatic reject conditions
- Core chain is not clear
- Start Focus is not visually obvious
- Secondary area is brighter than Next Action
- Mobile first viewport is occupied by secondary content
